Installation

Required packages

The required packages for Hexagn to compile are:
  • G++ (or Clang++ for MacOS Homebrew)
  • Make
If the required packages are not installed in your system, there are steps on how to install the required packages.
For linux, G++ and Make should be pre-installed by default, if not there are steps on how to install them.

Installing in Windows

The Installation of Hexagn in Windows is not available yet :(

Installing in MacOS

To install the required package, run the following commands in the terminal depending on your package manager

Using Homebrew 
$ brew update
$ brew install llvm # Clang++


Using MacPorts 
$ sudo port selfupdate
$ sudo port install gcc11 # GCC

Getting the compiler

To get the version of the repository, go to this page and download your desired version.
After installing the .zip file, extract it and go to the next step.

Installing Hexagn

After extracting the repository, go to the directory by using cd [directory] or access it via your file manager and opening in the terminal.

Run this command when you are in the directory 
$ make CXX=clang++  # Run this only if you installed via Homebrew
$ make              # Run this only if you installed via MacPorts


If you did all the process correctly, Hexagn should be installed

Check with 
$ hexagn --version


Installing in Linux

If the required packages are not pre-installed in your linux distro, run these command depending on your package manager:

APT - Debian, Ubuntu, Kali, etc 
$ sudo apt-get upgrade
$ sudo apt-get install g++


DNF/YUM - Fedora, RHEL, CentOS, OpenSUSE, etc 
$ sudo dnf upgrade
$ sudo dnf install gcc-c++

# For yum:
$ sudo yum upgrade
$ sudo yum install gcc-c++


PACMAN - Arch, Manjaro, etc 
$ sudo pacman -Syu
$ sudo pacman -Sy g++
