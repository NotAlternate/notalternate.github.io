Hexagn

A simple programming language

Hexagn is a simple easy to understand programming language that is compiled to URCL.
The syntax is very similar to the C++/Kotlin programming language, but simplier.
It is typed like C++/Kotlin/Javascript but simplier.
There are a lot of abstractions inside the language and the syntax.

See the simplicity of Hexagn from these examples:


int32 main() {
    // This example will add two numbers and prints them
    int8 a = 5;
    int8 b = 10;

    println("The sum of a and b is: ${a + b}");
}

Output 
The sum of a and b is: 15


Cross-Platform

Hexagn is compiled to URCL which is then emulated on platforms like:

  • PC (Windows, MacOS, Linux)
  • Web (Web browsers)
  • Mobile (Android, iOS)
